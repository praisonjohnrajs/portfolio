Featured Project

ConnectHub

Enterprise-grade real-time communication and call center management platform. ConnectHub unifies CRM, live agent monitoring, IVR management, and Asterisk-based telephony into a single powerful dashboard — enabling call centers to operate with maximum efficiency and real-time visibility.

  • Real-time agent dashboard with live call tracking across queues and locations
  • WebSocket + Redis Pub/Sub powered live status updates without page refresh
  • Asterisk AMI integration for complete call flow control and IVR management
  • Interactive donut charts showing agent states: On Call, Ready, Not Ready, Logout
  • Call listening and barging functionality for supervisor monitoring
  • JWT + 2FA/MFA authentication with role-based access control
  • Multi-language and location-wise agent grouping with live call counts
